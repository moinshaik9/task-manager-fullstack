\# Task Manager Full Stack App



\## Problem

Managing daily tasks manually is inefficient.



\## Features

\- Add tasks

\- View tasks

\- Mark as completed

\- Delete tasks



\## Tech Stack

\- Frontend: HTML, CSS, JavaScript

\- Backend: Node.js, Express

\- Database: MongoDB



\## How to Run

1\. npm install

2\. node app.js

3\. Open client/index.html
